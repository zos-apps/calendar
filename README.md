## About

Manage your schedule with this intuitive calendar application.

## Features

- Month/week/day views
- Event creation
- Reminders
- Multiple calendars
- Sync support

## Installation

### From App Store
Open the **App Store** in zOS and search for "Calendar".

### Via npm
```bash
npm install @zos-apps/calendar
```

## Usage

- Click date to add event
- Drag to reschedule
- Use arrows to navigate months
